## Tính năng

### Bộ gõ tiếng Việt
- Hỗ trợ **Telex** và **VNI**
- Phím **Z** phân rã tiếng Việt → gõ tiếng Anh không cần tắt bộ gõ
- Nhẹ nhàng, không chiếm tài nguyên

### Deep Work - Tập trung cao độ
- Chế độ tập trung 25 / 40 / 60 phút
- Nhắc nhở nhẹ nhàng mỗi 5 phút
- Thông báo khi còn 1/2 thời gian và 5 phút cuối
- Tooltip hiển thị task và thời gian còn lại

### Thư giãn (Mindfulness)
- Nhắc thư giãn mỗi 15 phút
- Gợi ý hít thở, giãn cơ, uống nước

### Âm thanh gõ phím
- 5 kiểu: Máy đánh chữ, Phím cơ, Bong bóng, Gõ nhẹ, Trống trầm
- Điều chỉnh âm lượng bằng Ctrl+Shift+Wheel

### Hoán đổi phím (cho lập trình viên)
- CapsLock → Tab
- Tab → ~
- ~ → CapsLock

## Phím tắt

| Phím tắt | Chức năng |
|----------|-----------|
| `Ctrl+Shift+S` | Bật/tắt bộ gõ |
| `Ctrl+Shift+D` | Bật/tắt Deep Work |
| `Ctrl+Shift+M` | Bật/tắt nhắc thư giãn |
| `Ctrl+Shift+A` | Đổi âm thanh gõ phím |
| `Ctrl+Shift+Wheel` | Tăng/giảm âm lượng |
| `Z` | Phân rã tiếng Việt |

## Cài đặt

1. Tải file **CozyKey_Setup_v1.4.0.exe**
2. Chạy file cài đặt (cần quyền Administrator)
3. Làm theo hướng dẫn trên màn hình
4. CozyKey sẽ chạy trong system tray
